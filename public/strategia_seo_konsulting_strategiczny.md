## 6. Checklist SEO Technicznego

### 6.1 Fundacja Techniczna

#### **Core Web Vitals Optimization**
- [ ] **Loading Performance (LCP < 2.5s)**
  - Optimize hero images (WebP format, lazy loading)
  - Minimize JavaScript execution time
  - Use efficient caching strategies
  - CDN implementation for static assets

- [ ] **Interactivity (FID < 100ms)**
  - Remove unused JavaScript
  - Code splitting dla large bundles
  - Optimize third-party scripts loading
  - Preload critical resources

- [ ] **Visual Stability (CLS < 0.1)**
  - Set explicit dimensions for images/videos
  - Avoid inserting content above existing content
  - Use font-display: swap for web fonts
  - Reserve space for ads/embeds
